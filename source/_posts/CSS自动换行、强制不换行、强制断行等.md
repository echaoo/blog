始终记不住强制不换行，记录一下：

##### 强制不换行：

```css
 white-space:nowrap;
```

##### 自动换行：

```css
 word-wrap:break-word;
```


##### 强制英文断行：

```css
 word-break:break-all;
```


##### 超出显示省略号：

```css
 ext-overflow:ellipsis;
 overflow:hidden;
```
